# My Game Weather

My Game Weather is a web application that allows users to view the weather forecast for the location of upcoming sports fixtures. Users can search for a team and view their next 5 fixtures, then click on a fixture to view the weather forecast for the fixture's location.

## Concept

The concept of My Game Weather is to provide sports fans with up-to-date weather information for upcoming fixtures. The app aims to improve the matchday experience for fans by providing them with important weather information to help them plan their day and stay comfortable while watching their favorite teams play.

The app uses the RapidAPI football API to retrieve information about upcoming fixtures and the OpenWeatherMap API to retrieve weather information for the fixture's location.

## Tech Stack
The tech stack used in My Game Weather is as follows:

* React: A popular JavaScript library for building user interfaces.
* Sass: A CSS preprocessor that allows for more efficient and modular styling.
* Axios: A promise-based HTTP client for making API calls.
* RapidAPI: A platform that allows developers to access and integrate with multiple APIs.
* OpenWeatherMap API: A weather API that provides weather data for cities and locations around the world.

## Usage

To use My Game Weather, follow these steps:

* Open the app in a web browser.
* Search for a team using the search box.
* Click on the team to view their next 5 fixtures.
* Click on a fixture to view the weather forecast for the fixture's location.
